Many codes are referenced in the Astrophysics Source Code Library (ASCL).

Interstellar and intergalactic cosmic-ray propagation - developed for studying steady sources

  • GALPROP
  • DRAGON
  • PICARD
  • CRPropa, CRPropa 3.1
  • SimProp

Cosmic-ray interactions and subsequent nuclear cascades

  • CRPropa
  • Monte Carlo code for UHECR, galaxy clusters
  • Monte Carlo code for UHECR, GRB internal shocks
  • NeuCosmA: code solving sets of partial differential kinetic equations. Other references: here, there.

Radiation of accelerated leptons and hadrons (mostly nucleons)

Most of these codes have been developed for multi-wavelength modeling of active galactic nuclei spectra.
Some of these codes are described and compared here and there.

  • AM3
  • ATHEνA
  • B13
  • LeHa-Paris
  • agnpy: open-source python package
  • NAIMA
  • JetSeT: Jets SED modeler and fitting Tool
  • Time-dependent code

Interaction cross sections and interaction products

  • SOPHIA: photopion production
  • TALYS: photonuclear interactions
  • EPOS: hadronic interactions
  • SIBYLL: hadronic interactions

Detection focussed codes

  • nuSpaceSim
  • nuPyProp: simulation of tau neutrino and muon neutrino interactions in the Earth. Open source.
  • Clsim: light in the deep glacial ice at the south pole or mediterranean sea water
  • Fennel: light from tracks and cascades
  • nuSQuIDS: neutrino propagation

Other codes

  • tbnew: modeling of interstellar absorption
